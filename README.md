# NetSweep 🔍

Herramienta de escaneo de red **más potente que nmap**, con detección de versiones, scripts de vulnerabilidad, escaneo WiFi y exportación de resultados.

## 🚀 Características
- Escaneo de red local (ARP, ICMP, TCP SYN)
- Detección de versiones de servicios (HTTP, SSH, FTP, SMB, etc.)
- Scripts de vulnerabilidad integrados
- Escaneo de redes WiFi (redes cercanas + dispositivos conectados)
- Exportación a JSON, CSV, HTML y PDF
- Cross-platform: Windows y Linux
